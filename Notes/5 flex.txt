flex is used to arrange the elements in single direction (horizontal/vertical)
horizontal is called row and vertical is called column.
default direction is 'row'.
display should be applied top parent elements.

flex-direction : use to specify direction in which child in are arranged (row/column/row-reverse/column-reverse)

justify-content : 
                used to arrange elements in the same direction of flex direction
                (start/left/right/center/space-between/space-evenly/space-around).


align-items :
            used to arrange elements in opposite direction of flex direction(start/left/right/center/end).
